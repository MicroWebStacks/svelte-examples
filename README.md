# svelte-examples
examples with svelte

supported auto deployments
* Cloudflare Pages
* Netlify
* Vercel
* Azure Static Web Apps

adapters
* adapter-static
* adapter-node
* adapter-netlify
* adapter-cloudflare
* adapter-cloudflare-workers
* adapter-vercel

community (deno, github, do, bun)
